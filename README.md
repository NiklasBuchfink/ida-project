## Getting Started

For starting locally install first all the dependencies by typing in the console:

```bash
npm install
# or
yarn install
```

Then, to run the development server, type:

```bash
npm run dev
# or
yarn dev
```

Finally open [http://localhost:3000](http://localhost:3000) with your browser.
